TIP BOX의 기본 margin값은 16px 0입니다. 그러나 페이지별로 상이할 수 있으니, 슬랙에 공유드리는 [_spacingVer2.css] 파일을 참고하셔서 클래스네임을 추가해 주시기 바랍니다.
예시는 아래와 같습니다.

mgt_{n} : { margin-top : {n}px; }
mgb_{n} : { margin-bottom : {n}px; }
mgl_{n} : { margin-left : {n}px; }
mgr_{n} : { margin-left : {n}px; }
mgt_{n} : { margin-top : {n}px; }

페이지마다 text-align이 left인 것이 있고 center인 것이 있으니 아래 클래스네임 참조 부탁드립니다.
